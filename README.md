Program Hakkında:

Bu program kullanıcıdan alınan girdinin ilk karakterini "ASCII" kodlarını kullnarak yorumlamakta ardından karakter türünü ekranda görüntülemektedir,programın yapabildiği sınıflandırmalar aşağıdaki gibidir:

a) "UPPER" (Büyük Harf)

b) "LOWER" (Küçük Harf)

c) "DIGIT" (Rakam)

d) "SPACE" (Boşluk Karakteri)

e) "PUNCT" (Noktalama İşareti)

f) "OTHER" (Diğer Karakterler)

Nasıl Çalıştırılır?

1)Program çalıştığında "ASCII" koduna göre yorumlamak için sizden bir girdi beklemektedir

2)Lütfen bir girdi giriniz(programa herhangi karakterden oluşan bir girdi yazabilirsiniz).

3)Girdinizin ilk karakteri program tarafından "ASCII" koduna çevirilerek sınıflandırılır.

4)Yapılan sınıflandırmanın sonrası elde edilen sonuc ekrana yazdırılır.
